# Loan Prediction System

## 📌 Overview
The Loan Prediction System is a machine learning-based web application that predicts whether a loan should be approved or not based on user input details. It helps financial institutions make faster and more accurate decisions.

This project uses a trained machine learning model and a user-friendly interface to provide real-time predictions.

---

## 🚀 Features
- Predict loan approval status instantly
- User-friendly web interface
- Machine Learning model using Random Forest
- Handles multiple input features
- Fast and efficient predictions

---

## 🛠️ Technologies Used
- Python
- Flask
- HTML/CSS
- Pandas
- Scikit-learn
- Pickle (for model saving)

---

## 📊 Machine Learning Model
- Algorithm: Random Forest Classifier
- Dataset: Loan dataset (CSV file)
- Model trained using features like:
  - Income
  - Loan Amount
  - Credit History
  - Employment status
  - Other financial details
